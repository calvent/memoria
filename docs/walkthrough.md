## 使用说明

### 数据填充步骤

1. **准备环境**
   ```bash
   cd /Users/liaochui/code/memoria/backend-service-py
   # 确保数据库已启动
   # 确保已安装依赖 (uv sync 或 pip install -r requirements.txt)
   ```

2. **运行seed脚本**
   ```bash
   # 直接填充数据
   python seed_data.py
   
   # 清空现有数据后填充(谨慎使用)
   python seed_data.py --clean
   
   # 仅验证数据
   python seed_data.py --verify
   ```

3. **验证结果**
   
   脚本会自动输出:
   - 创建进度
   - 数据统计
   - 验证结果

### API测试

```bash
# 查看回忆录列表
curl http://localhost:8999/api/v1/memoir/list

# 查看回忆录详情
curl http://localhost:8999/api/v1/memoir/detail?memoir_id=1

# 查看章节列表  
curl http://localhost:8999/api/v1/chapter/list?memoir_id=1
```

### 前端验证

1. 启动后端服务: `cd backend-service-py && npm run dev`
2. 启动前端小程序: `cd frontend && npm run dev:weapp`
3. 进入"回忆录广场"页面
4. 确认能看到4个回忆录
5. 点击查看详情,验证章节内容
