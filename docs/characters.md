# 人物档案设计文档

本文档记录4位主要人物的人生轨迹设计,基于真实历史背景创建。

## 1. 张天明 - 知青返城后的普通工人

### 基本信息
- **姓名**: 张天明
- **出生**: 1950年 (虚岁74岁)
- **籍贯**: 上海市杨浦区
- **职业经历**: 知青 → 工厂工人 → 退休

### 人生轨迹

#### 童年时期 (1950-1966)
- 1950年出生于上海杨浦区工人家庭
- 父亲是码头工人,母亲在纺织厂工作
- 在里弄长大,有两个弟弟
- 1962年进入中学,是"老三届"学生(66、67、68届)

#### 知青岁月 (1968-1978)
- 1968年,18岁时响应号召下乡到安徽农村插队
- 在农村劳动10年,从不会农活到成为生产队骨干
- 1977年高考恢复,因文化底子薄未能考上
- 1978年,28岁时通过招工政策返回上海

#### 工作生涯 (1978-2010)
- 1978年进入上海某机械厂当工人
- 从学徒工做起,逐步成长为技术工人
- 1985年结婚,妻子也是厂里的工人
- 1987年儿子出生
- 1995年厂里效益下滑,经历减薪但未下岗
- 2010年60岁正式退休

#### 退休生活 (2010-至今)
- 退休后帮忙带孙子
- 喜欢下棋、散步
- 偶尔和老工友聚会

## 2. 李秀芳 - 扎根乡村40年的小学教师

### 基本信息
- **姓名**: 李秀芳
- **性别**: 女
- **出生**: 1952年 (虚岁72岁)
- **籍贯**: 江苏苏州农村
- **职业**: 民办教师 → 公办教师 → 退休

### 人生轨迹

#### 童年与求学 (1952-1970)
- 1952年出生于苏州郊区农村
- 父母都是农民,家中排行老大
- 小学成绩优异,初中毕业后因家境未能继续升学
- 1970年18岁,高中毕业

#### 初为人师 (1970-1986)
- 1970年被村里推荐为民办教师
- 在村小学教语文和算术
- 工资由生产队记工分,非常微薄
- 白天教书,晚上还要参加农业劳动
- 1975年结婚,丈夫是邻村的赤脚医生
- 1978、1982年两个孩子出生

#### 转正与教改 (1986-2010)
- 1986年通过考试转为公办教师
- 工资待遇大幅提升,家庭生活改善
- 1990年代参与教育改革,学习新的教学方法
- 培养了数百名学生,桃李满天下
- 2000年后学校撤点并校,到镇中心小学任教

#### 退休生活 (2010-至今)
- 2010年58岁退休(女教师退休年龄)
- 经常收到学生的探望和问候
- 帮助照看孙辈
- 参与村里老年活动

## 3. 王建国 - 从赤脚医生到主任医师

### 基本信息
- **姓名**: 王建国
- **出生**: 1955年 (虚岁69岁)
- **籍贯**: 上海市黄浦区
- **职业**: 知青/赤脚医生 → 医学院学生 → 医生 → 主任医师 → 退休

### 人生轨迹

#### 童年求学 (1955-1973)
- 1955年出生于上海,父亲是中学教师
- 1973年高中毕业,正值上山下乡高峰

#### 下乡与赤脚医生 (1973-1977)
- 1973年18岁下乡到江西农村
- 1974年被选送参加赤脚医生培训
- 在农村卫生所工作,服务乡亲

#### 重返校园 (1977-1982)
- 1977年恢复高考,以优异成绩考入上海第二医科大学
- 1982年大学毕业,27岁

#### 医者生涯 (1982-2015)
- 1982年分配到上海某三甲医院内科
- 从住院医生做起
- 1990年晋升主治医师
- 1998年晋升副主任医师
- 2005年晋升主任医师
- 见证了中国医疗卫生事业的巨大发展
- 经历了医疗改革的各个阶段

#### 退休生活 (2015-至今)
- 2015年60岁退休
- 偶尔在社区医院参与义诊
- 关注医疗健康话题

## 4. 刘和平 - 钢铁工人的荣光与转型

### 基本信息
- **姓名**: 刘和平
- **出生**: 1953年 (虚岁71岁)
- **籍贯**: 上海市浦东(当时属川沙县)
- **职业**: 学徒工 → 钢铁工人 → 高级技师 → 退休

### 人生轨迹

#### 童年成长 (1953-1970)
- 1953年出生,父亲是钢铁厂老工人
- 在工人新村长大
- 初中文化程度

#### 学徒岁月 (1970-1975)
- 1970年17岁,通过父亲介绍进入上钢三厂当学徒工
- 跟随师傅学习炼钢技术
- 工作环境艰苦,高温噪音
- 但对工人身份感到自豪

#### 技术成长 (1975-1995)
- 1975年出师,成为正式工人
- 1978年结婚,妻子在纺织厂工作
- 1980年儿子出生
- 1985年通过技术考核,晋升为技师
- 参与技术革新,获得厂里表彰
- 1990年晋升高级技师

#### 改革阵痛 (1995-2010)
- 1995年后钢铁厂效益下降
- 2000年工厂搬迁改制
- 幸运的是作为技术骨干保留岗位
- 工资待遇有所下降,但坚持到退休

#### 退休生活 (2010-至今)
- 2010年57岁退休(特殊工种提前退休)
- 和老工友们保持联系
- 见证了上钢三厂旧址变成世博园区
- 对那段岁月既怀念又感慨

## 注意事项

1. **时间线要准确**: 每个人物的年龄、关键事件时间点要与历史吻合
2. **地域特色**: 体现上海及周边地区的特点
3. **语言风格**: 回忆内容要符合人物身份和时代特征
4. **敏感内容**: 适度提及历史背景,不过分渲染
5. **情感真实**: 体现普通人的喜怒哀乐,不刻意拔高
